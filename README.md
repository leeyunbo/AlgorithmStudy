# Algorithm Study

### 레포지토리 소개 
```
다양한 Algorithm 학습을 위한 레포지토리 
기본적인 알고리즘 학습 후, 다양한 활용 문제를 풀며 Study 진행 
문제를 푼 후, 다른 사람들의 코드를 확인한 후 효율성에 문제가 발견되면 수정 
```
### 목표
```
기본에 충실하기 
익숙해지기 
생각할 줄 알기 
```
